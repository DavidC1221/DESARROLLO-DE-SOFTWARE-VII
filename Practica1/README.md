# Práctica PHP - Estructuras de Control

Este repositorio contiene la solución de 3 problemas básicos en PHP:

1. Operaciones matemáticas y precedencia
2. Determinar si un número es par o impar
3. Tabla de multiplicar usando ciclo for

## Requisitos
- XAMPP o WAMP
- Navegador web

## Ejecución
Colocar la carpeta en:
htdocs (XAMPP)
